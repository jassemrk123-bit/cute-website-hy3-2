Photo gallery

No memories yet. Add items to window.SiteConfig.gallery.items in assets/js/config.js.

How to add pictures

  1. Create an assets/photos folder.
  2. Drop your image files inside, for example us-1.jpg.
  3. Open assets/js/config.js and add an item with src: "assets/photos/us-1.jpg".
  4. Refresh the page. The gallery updates itself.
