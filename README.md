# For You — A Premium Keepsake Website

This is a small, elegant static website created as a personal gift.
It is built with plain HTML, CSS, and JavaScript so it can grow slowly
without requiring a framework.

## What is included

- `index.html` — the landing experience with an envelope, letter, and doves.
- `gallery.html` — a photo gallery page with a lightbox.
- `assets/css/style.css` — the core design system and components.
- `assets/css/theme.css` — alternative color moods.
- `assets/js/config.js` — edit text, doves, gallery photos, and easter eggs here.
- `assets/js/experience.js` — envelope, dove animation, and theme toggle.
- `assets/js/gallery.js` — gallery rendering and lightbox.
- `assets/js/easterEggs.js` — hidden interactions.
